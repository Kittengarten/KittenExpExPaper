# KittenExpExPaper

## 简介

KittenExpExPaper 是一个 Minecraft Java 版`Paper`插件，由`Java 19`编译，仅支持`1.19`以上版本。

## 功能

用经验值交换一些资源

* 使用命令`/expex list`查询可交换的资源列表及其命令
* 使用命令`/expex item [命名空间 ID] [数量]`交换对应资源
* 如因为背包已满，未能获取资源，不予处理
